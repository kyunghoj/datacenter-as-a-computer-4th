# Datacenter as a Computer, 4th Edition

[Datacenter as a Computer, 4th Edition](https://link.springer.com/book/10.1007/978-3-031-99489-0)의 한국어 번역입니다. `gemini-3-pro-preview` 를 이용하여 PDF 에서 텍스트를 뽑아내고 한국어로 번역했습니다.

## How to build

아래와 같이 [MkDocs](https://www.mkdocs.org/getting-started/)를 이용하여 HTML 문서를 만들 수 있습니다.

```shell
python3 -m venv .venv
source .venv/bin/activate
pip install mkdocs
mkdocs serve
```
